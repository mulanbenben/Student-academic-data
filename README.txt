The datasets contain below files: 

1.Data_Course.txt: this file contains the year and semester in which the course set up, ID of student who selects the course, course ID, ID of teacher who teaches the course, and course credit.
2.Data_Exam.txt: this file contains year and semester in which the student attends course, course ID, course name, ID of student who takes the test, first takes/retakes exam, exam score, and course credit.
3.Data_Schoolarship.txt: this file contains the amount of each scholarship awarded, semester (expressed in years) and ID of student who obtains scholarship.
4.Data_Campus_card.txt: this file contains ID of student whose record, meal time and consumption amount.
